## 🌟 **Nombre de la meta:** Texto a storyboard visual

**Descripción breve:** Crear un sistema de inteligencia artificial que transforme descripciones textuales en una **secuencia visual narrativa (storyboard)**, utilizando modelos generativos y procesamiento de lenguaje natural (NLP).  

Este proyecto servirá tanto como **prueba de habilidades técnicas** para Sakana AI como para expresar creativamente emociones y vivencias personales en forma visual.  

📅**Fecha de inicio:** 2025-05-25 
📅**Fecha límite estimada:** 2025-06-30

| **Hito**  | **Nombre**                            |
| --------- | ------------------------------------- |
| ⬜ 🟩**1** | Análisis de Texto y Emociones (NLP)   |
| ⬜ 🟨**2** | Generación de Personajes y Escenarios |

| ⬜ 🟥**4** | Optimización y Publicación            |
### 🟩 ** 1: Análisis de Texto y Emociones (NLP)**
📅 **Fecha objetivo:** 2025-05-30

**Descripción:**  
Desarrollar un módulo que analice el texto ingresado por el usuario y detecte emociones, contexto y posibles elementos visuales clave.  

✅ **Acciones clave:**  
- [ ] Investigar modelos de **Procesamiento de Lenguaje Natural (NLP)** adecuados.  
- [ ] Implementar un modelo que analice emociones y extraiga palabras clave.  
- [ ] Validar resultados con ejemplos de descripciones reales.  

---

### 🟨 ** 2: Generación de Personajes y Escenarios**  
📅 **Fecha objetivo:** 2025-06-15

**Descripción:**  
Desarrollar modelos de IA que generen personajes y escenarios visuales basados en el análisis de emociones y contexto textual.  

✅ **Acciones clave:**  
- [ ] Explorar modelos generativos de imágenes como **Stable Diffusion** y **GANs**.  
- [ ] Implementar un modelo que genere **personajes específicos** según el contexto del texto.  
- [ ] Implementar un modelo que genere **fondos y escenarios** adecuados.  
- [ ] Probar con diferentes prompts y ajustar parámetros.  

---

### 🟧 ** 3: Integración de Elementos Visuales y Secuenciación**  
📅 **Fecha objetivo:** 2025-07-07

**Descripción:**  
Unir los módulos anteriores para crear un flujo de trabajo completo: texto → análisis → generación de imágenes → storyboard.  

✅ **Acciones clave:**  
- [ ] Diseñar el **flujo de datos** entre los módulos de texto, personajes y escenarios.  
- [ ] Implementar un integrador que combine personajes y escenarios en una narrativa visual coherente.  
- [ ] Ajustar estilos visuales y probar variaciones de animación.  

---

### 🟥 ** 4: Optimización, Documentación y Presentación Final**  
📅 **Fecha objetivo:** 2025-05-24  

**Descripción:**  
Mejorar la calidad del sistema, documentar todo el proceso y publicar el repositorio en GitHub para recibir retroalimentación.  

✅ **Acciones clave:**  
- [ ] Optimizar modelos y mejorar eficiencia computacional.  
- [ ] Redactar documentación clara y detallada (README, notebooks, ejemplos).  
- [ ] Crear una presentación visual explicando el proyecto y sus aplicaciones.  
- [ ] Publicar en GitHub y recibir feedback de la comunidad técnica.
